# FtpReaderParser

Ruby gem to enter in a ftp service, read csv document and get parsed json object 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ftp_reader_parser'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ftp_reader_parser
